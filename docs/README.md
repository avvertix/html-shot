# Documentation

Guides for rendering HTML to images with `html-shot`.

- **[Getting started](getting-started.md)** — install the package and the native
  library, then render your first image.
- **[Fonts & styles](fonts-and-styles.md)** — load custom fonts and style your
  markup with inline CSS, `<style>` blocks, external stylesheets, or `tw`
  utilities.
- **[Advanced usage](advanced.md)** — HiDPI / Retina output, and driving a
  `Context` + `Renderer` directly for repeated rendering.

For complete, runnable scripts see the [`examples/`](../examples/) directory and
its [README](../examples/README.md). The main [project README](../README.md) has
a "Deep Dive" section describing how the engine interprets HTML and CSS.
